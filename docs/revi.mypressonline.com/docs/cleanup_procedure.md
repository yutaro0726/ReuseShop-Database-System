# ファイル整理作業手順書
作成日：2025-06-18

## 1. 事前確認事項

### 1.1 バックアップ作業
1. データベースの完全バックアップを取得
   ```sql
   mysqldump -u 4646140_revi -p 4646140_revi > backup_20250618.sql
   ```

2. 重要ディレクトリのバックアップ
   - `/system`ディレクトリ
   - `/admin`ディレクトリ
   - 設定ファイル類

### 1.2 動作確認項目
- [ ] 管理画面ログイン
- [ ] 各管理機能の表示確認
- [ ] レビュー投稿・編集機能
- [ ] データベース連携確認
- [ ] Basic認証の動作確認

## 2. ファイル削除手順

### 2.1 管理画面CSS関連
1. 新CSSファイルの動作確認
   - `/admin/assets/css/admin.css`の内容確認
   - 全画面での表示確認

2. 古いCSSファイルの削除
   ```
   rm old/admin/admin.css
   rm -rf old/admin/base/*
   ```

### 2.2 一時ファイル
1. データベース同期確認後に削除
   ```
   rm /tmp/mysqldump.sql
   ```

2. 同期スクリプトの削除
   ```
   rm system/sync_script.php
   ```

### 2.3 バックアップファイル
- バックアップの必要性を確認後、以下を削除
   ```
   find . -name "*.bak" -type f -delete
   find . -name "*.backup*" -type f -delete
   ```
