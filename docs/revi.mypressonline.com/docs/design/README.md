# Design ディレクトリ

このディレクトリには、システムの設計資料を格納します。

## 使用方法
- システム設計図、ER図、フローチャート等を格納
- Mermaid記法やDiagrams.net（draw.io）形式のファイルを推奨
- 設計内容を元にコーディングを実施

## ファイル形式
- Markdownファイル（.md） - Mermaid記法を含む
- Draw.ioファイル（.drawio）
- 画像ファイル（.png, .jpg, .svg等）

## 設計ファイルの種類
- システム相関図
- データベースER図
- 画面遷移図
- API設計書
- UI/UXワイヤーフレーム

## 注意事項
- 設計内容と実装が乖離しないよう、設計変更時は図解も必ず更新
- 設計内容を元にコーディングを行う
- 生成AIが読み書きすることを前提とした記述
