# Sample ディレクトリ

このディレクトリには、外部サイトのサンプル等を格納します。

## 使用方法
- 外部サイトのサンプルが必要になった場合、HTML形式などで格納
- 必要に応じて参照
- デザインやUI/UXの参考資料として活用

## ファイル形式
- HTMLファイル（.html）
- CSSファイル（.css）
- JavaScriptファイル（.js）
- 画像ファイル（.jpg, .png, .gif等）

## 注意事項
- 外部からのサンプルファイルには著作権やライセンスに注意
- 参考用途に留め、そのまま本番環境で使用しない
